*DISCRIPCTION OF PROJECT* = # Iris Flower Classification using Machine Learning
This project is a beginner-friendly Machine Learning project that classifies iris flowers into different species using supervised learning algorithms. The project uses the famous Iris dataset, which contains flower measurements such as sepal length, sepal width, petal length, and petal width.
The main objective of this project is to predict the species of an iris flower based on these measurements. The three target classes in the dataset are:
* Iris-setosa
* Iris-versicolor
* Iris-virginica
In this project, two Machine Learning algorithms are implemented:
1. Decision Tree Classifier
2. k-Nearest Neighbors (k-NN)
The dataset is loaded using Pandas, and the features and target labels are separated. After preprocessing, the data is split into training and testing sets using an 80:20 ratio. The models are then trained using Scikit-learn.
After training, predictions are made on the test dataset, and the performance of each model is evaluated using:
* Accuracy Score
* Precision
* Recall
* F1-Score
* Classification Report
The Decision Tree algorithm works by creating decision rules based on feature values, while the k-NN algorithm predicts the class by finding the nearest neighboring data points.
This project helps beginners understand important Machine Learning concepts such as:
* Data preprocessing
* Train-test splitting
* Model training
* Prediction
* Performance evaluatioN
Technologies used in this project include:
* Python
* Pandas
* Scikit-learn
The project demonstrates how Machine Learning models can accurately classify data and solve real-world prediction problems. It is a simple and effective project for students and beginners who want to learn the basics of classification algorithms and supervised learning.
